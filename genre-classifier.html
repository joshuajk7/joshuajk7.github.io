Description

This project implements a convolutional neural network (CNN) for classifying music into genres based on extracted audio features. The model uses Mel-Frequency Cepstral Coefficients (MFCCs) for input and is trained and evaluated using TensorFlow and Keras.

Technologies Used

  • Python
  • TensorFlow
  • Keras
  • NumPy
  • Librosa
  • Scikit-learn
  • Matplotlib
