Description

This is a full-stack administrative application built for Global Bread Co., enabling management of customer orders, product inventory, and delivery tracking. The app features secure role-based access, a responsive frontend, and a well-structured backend for scalability and reliability.

Technologies Used

  • Python
  • Flask
  • HTML/CSS/JavaScript
  • MySQL / MariaDB
  • RESTful API
  • Jinja2
  • Cloud Deployment
